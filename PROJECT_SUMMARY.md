### Common Issues

**Build Error**: `Module not found`
```bash
rm -r .next node_modules
npm install
npm run build
```

**Port 3000 in use**:
```bash
npm run dev -- -p 3001
```

**Styling not loading**:
- Clear browser cache (Ctrl+Shift+Delete)
- Restart dev server
- Check tailwind.config.js
